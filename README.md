# Kioku Releases

Public release downloads and Sparkle auto-update feed for [Kioku](https://github.com/joerodriguez/kioku).

## Appcast

The Sparkle appcast is served at:
```
https://raw.githubusercontent.com/joerodriguez/kioku-releases/main/appcast.xml
```

This URL is baked into `SUFeedURL` in the app's `Info.plist`.

## Installing

Download the latest `.dmg` from the [Releases](https://github.com/joerodriguez/kioku-releases/releases) page.
